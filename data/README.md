# Data

- `Proj3613.csv` — conjoint responses for **protein** and **grain** factors.
- `vegetable.csv` — conjoint responses for the **vegetable** factor.

The raw survey export (`Eating habits 2.csv`) used for the PCA, clustering and
regression steps is **not included** because it contains personal respondent
information. Supply your own copy and update `SURVEY_PATH` in
`../code/01_pca_clustering.R` to reproduce those steps.
